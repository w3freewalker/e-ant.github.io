解决方案：

设置MySQL随机自启动

  1. 确保/etc/rc.d/init.d/mysql存在

    若用service mysql start能够正常启动，表示服务已存在(若是通过rpm方式安装的，相应服务会自动在/etc/rc.d/init.d目录注册的)

  2. 检查在所有linux运行级别中，系统随机启动服务项的运行状态信息(on 或 off)

  3. 若未找到,添加服务

  4. 设置相应运行级别开启随机启动功能

  5. 重启mysql服务
